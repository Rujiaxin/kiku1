1、git增加要提交的目录
  git add html/texiao.html

2、git提交代码
  git commit -m "版本描述"

3、git的历史记录
  git log

  简易输出
  git log --pretty=oneline

4、退回到上个版本
  git reset --hard HEAD^
 上一个版本就是HEAD^，上上一个版本就是HEAD^^，当然往上100个版本写100个^比较容易数不过来，所以写成HEAD~100。

5、回到指定的版本
   git reset --hard 1094a（某个版本的commit id是1094adb...）

6、执行的每一条命令
   git reflog

7、撤销修改
   git checkout -- readme.txt

8、已添加git add 但未提交 git commit之前，取消修改
   git reset HEAD readme.txt

9、删除文件
   git rm test.txt
   git commit -m "remove test.txt"

10、提交至gitHub
	git push origin master
